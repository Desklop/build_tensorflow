Запустить контейнер (-t - запуск терминала, -i - интерактивный режим, --rm - удалить контейнер при завершении работы запускаемого приложения, -p 5000:5000 - пробросить все подключения на порт 5000 к машине-хосту в контейнер на порт 5000, --cpuset-cpus="0" - запустить на ядре "0"):
sudo docker run -ti --rm -v "$PWD:/mnt" -e HOST_PERMS="$(id -u):$(id -g)" build_tensorflow_1.13.1:0.1 bash

sudo docker run --cpuset-cpus="0-11" -m 8GB -ti --rm -v "$PWD:/mnt" -e HOST_PERMS="$(id -u):$(id -g)" build_tensorflow_1.13.1:0.1
sudo docker run --cpuset-cpus="0-11" -m 8GB --runtime=nvidia -ti --rm -v "$PWD:/mnt" -e HOST_PERMS="$(id -u):$(id -g)" build_tensorflow_gpu_1.13.1:0.1

Собрать образ с меткой build_tensorflow_1.13.1:0.1 на основе Dockerfile (. - директория, из которой вызывается docker build, точка - значит в текущей директории находятся все файлы для образа):
sudo docker build -t build_tensorflow_1.13.1:0.1 .
sudo docker build -t build_tensorflow_gpu_1.13.1:0.1 .

Сохранить образ вместе с внесёнными в него изменениями в .tar:
sudo docker save build_tensorflow_1.13.1:0.1 > build_tensorflow_1.13.1.tar
или
sudo docker save build_tensorflow_1.13.1:0.1 -o build_tensorflow_1.13.1.tar

Что бы распаковать образ (находясь с ним в одной директории):
sudo docker load < build_tensorflow_1.13.1.tar
или
sudo docker load --input build_tensorflow_1.13.1.tar

Переименовать образ:
sudo docker tag 5e5bef1bdb53 build_tensorflow_1.13.1:0.1

Сохранить контейнер в .tar:
sudo docker export fd5e8b6eeb04 -o build_tensorflow_1.13.1.tar

Загрузить контейнер из .tar:
sudo docker import build_tensorflow_1.13.1.tar

Удалить образ:
sudo docker rmi build_tensorflow_1.13.1:0.1
или
sudo docker rmi d3e0c6c72e78

Остановить контейнер:
sudo docker stop d3e0c6c72e78

Запуск остановленного контейнера:
sudo docker start d3e0c6c72e78

Подключиться к контейнеру:
sudo docker attach d3e0c6c72e78

Перезагрузка контейнера:
sudo docker restart d3e0c6c72e78

Удалить контейнер:
sudo docker rm d3e0c6c72e78

Список работающих контейнеров:
sudo docker ps -a

Список образов:
sudo docker images

Удалить все контейнеры и образы, очистить кеш:
sudo docker system prune -a
